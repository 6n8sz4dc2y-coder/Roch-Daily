# RRG Dealer Operating System v9.4

Upload `index.html` to the root of your GitHub Pages repo.

## Changes in v9.4

- Trade GP now feeds into the Used Cars GP forecast because trade sits within the used car target.
- Trade remains visible as a separate line for units and GP visibility.
- Vehicle Sales target no longer double-counts a separate trade target.
- Added Service Other Income from the DOC as a separate line.
- Service Other Income now counts in Aftersales Forecast and Total GP.
- Aftersales target now includes Service Other Income target.
- Forecast totals have been tightened so Sales, Aftersales, Total GP and Net Profit align more closely to the DOC structure.


## v9.4
- Vehicle Performance now splits unit pacing into Current RR, Required RR and Forecast RR.
- Required RR uses remaining days in the month rather than the current month-to-date pace.
- Forecast RR shows the pace implied by the management forecast for the remaining days.


## v9.4 changes
- Added top-row Enquiry Projection card.
- Compares projected sales from enquiries against management New + Used forecast.
- Reordered top cards so Vehicle Sales, Enquiry Projection, F&I and Stock Risk sit together, with Aftersales after them.
