# RRG Dealer Operating System v6.5

Upload `index.html` to the root of your GitHub Pages repo.

## What changed in v6.5

- Fleet line is now treated as **Motability**.
- Added Expected Motability Deliveries management input.
- Motability affects vehicle GP but does **not** increase F&I forecast.
- Manual New / Motability / Used inputs now update Vehicle Sales GP, Financial Performance, Total GP, Net Profit, Centre Pulse and Daily Update.
- F&I GP remains DOC-led; F&I log remains product/lender headlines only.
- Added Car Sales Trade line with units, GP per unit and forecast GP.
- Added optional Expected Trade Deliveries input.

## Daily process

1. Select Rochdale or Silsden.
2. Upload DOC.
3. Upload optional F&I, stock, funnel and ASPI reports.
4. Enter management forecasts for new, motability, used and trade deliveries if run-rate is misleading.
5. Add optional service / parts forecast adjustments.
6. Review Daily Focus and copy the update.
